# 🤖 ai-quickprojects

> 20 dias. 20 projetos. De zero a agentes de IA funcionais.

Um plano intensivo de aprendizagem prática onde construí um projeto de Inteligência Artificial por dia — do simples "Hello World" com a API do Claude até pipelines completos com RAG, tool use, CI/CD e automação visual.

---

## 🎯 O que aprendi

- **Agentes não são magia** — são loops simples com acesso a ferramentas. Assim que percebi isso, o resto ficou muito mais claro.
- **FastAPI + Railway = produção em minutos** — o ciclo de deploy ficou tão rápido que parei de ter medo de colocar coisas online.
- **GitHub Actions muda completamente os hábitos de código** — quando o lint e os testes correm automaticamente, a qualidade sobe sem esforço consciente.
- **RAG é mais sobre chunking do que sobre embeddings** — passei metade do Dia 13 a perceber que o problema não era o modelo, era como eu dividia o PDF.
- **Construir todos os dias bate qualquer curso** — ao fim de 20 dias tinha mais confiança prática do que meses de tutoriais passivos.
